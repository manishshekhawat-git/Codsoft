• Task 3: Iris Flower Classification

• Overview
This project classifies Iris flowers into three species using machine learning techniques.

• Dataset
The Iris dataset contains measurements of flowers:
- Sepal length
- Sepal width
- Petal length
- Petal width

• Species classified:
- Setosa
- Versicolor
- Virginica

• Machine Learning Model
A classification algorithm is used to predict the species of Iris flowers based on their measurements.

• Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn

• Outcome
The model accurately classifies Iris flowers into their respective species.
