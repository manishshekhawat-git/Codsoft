## Task 4: Sales Prediction Using Python

• Overview
This project predicts future sales based on advertising expenditure using machine learning regression techniques.

• Dataset
The dataset includes advertising spend on different platforms such as:
- TV
- Radio
- Newspaper

The target variable is sales.

• Machine Learning Model
Linear Regression is used to predict sales based on advertising costs.

• Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn

• Outcome
The model predicts sales effectively and helps businesses understand the impact of advertising on sales.
