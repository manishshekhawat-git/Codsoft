# CodSoft Data Science Internship

This repository contains the tasks completed as part of the CodSoft Data Science Internship program.  
The projects demonstrate beginner-level understanding of data analysis, machine learning, and Python programming.

---

##  Tasks Included

###  Task 1: Titanic Survival Prediction
- Predicts whether a passenger survived the Titanic disaster.
- Uses passenger data such as age, gender, class, and fare.
- Machine Learning Model: Logistic Regression

 Folder: `Task-1-Titanic-Survival-Prediction`

---

###  Task 3: Iris Flower Classification
- Classifies Iris flowers into three species:
  - Setosa
  - Versicolor
  - Virginica
- Uses flower measurements like sepal and petal length.
- Machine Learning Model: Classification Algorithm

 Folder: `Task-3-Iris-Flower-Classification`

---

###  Task 4: Sales Prediction Using Python
- Predicts sales based on advertising expenditure.
- Uses data from TV, Radio, and Newspaper advertising.
- Machine Learning Model: Linear Regression

 Folder: `Task-4-Sales-Prediction`

---

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

---

##  How to Run the Projects
1. Install required libraries from `requirements.txt`
2. Run the Python file inside each task folder
